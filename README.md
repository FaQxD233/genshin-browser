# Genshin Browser

一个专门为 B 站原神攻略视频设计的 Windows 浮窗浏览器，基于 `WPF + WebView2`。

## ✨ 核心特性

### 🎮 游戏友好
- **固定模式**：窗口置顶，锁定位置，完美搭配游戏使用
- **全局快捷键**：按 `K` 键控制视频播放/暂停，按 `F8` 切换模式
- **管理员权限**：提高全局热键在游戏环境下的响应率

### 🌐 浏览体验
- **登录态保留**：固定复用 WebView2 profile，B 站登录态、Cookie 和缓存永久保存
- **智能恢复**：记住上次浏览的页面、窗口位置和大小
- **实时地址同步**：控制窗口地址栏实时跟随浏览器 URL 变化
- **平滑滚动**：优化的滚动动画，流畅跟手

### 📋 管理功能
- **收藏夹**：一键收藏当前页面，支持实时搜索
- **浏览历史**：自动记录访问历史，支持实时搜索
- **右键菜单**：打开、复制链接、删除记录
- **清空历史**：一键清空所有浏览记录

### 🎨 现代 UI
- **GitHub 深色主题**：舒适的深色配色
- **双窗口设计**：主浏览器窗口专注显示内容，控制窗口提供所有操作
- **智能时间显示**：今天、昨天、相对时间智能格式化
- **响应式交互**：悬停缩放、淡入淡出、焦点反馈、状态指示
- **Toast 提示**：关键成功/失败状态以非阻塞提示展示

## 🖥️ 运行要求

- Windows 10/11
- Edge WebView2 Runtime（Windows 11 自带）
- .NET 8 Runtime（自包含版本无需安装）

## 📦 发布与使用

### 自包含版本（推荐）
- 发布脚本：`release/publish-win-x64.ps1`
- 发布目录：`dist/win-x64-self-contained/`
- 无需安装 .NET，开箱即用

### 运行
1. 下载并解压发布包
2. 运行 `GenshinBrowser.exe`
3. 首次启动会自动打开控制窗口

### 快捷键
- `K`：播放/暂停视频（全局）
- `F8`：切换固定/自由模式（全局）

## 🏗️ 架构特点

### 窗口管理
- **主浏览器窗口**：648x400，纯净的浏览体验
- **控制窗口**：640x853，自由模式下显示，提供所有控制功能

### 数据存储
- 配置文件：`%LocalAppData%\GenshinBrowser\settings.json`
- 浏览历史：`%LocalAppData%\GenshinBrowser\history.json`（最多 50 条）
- 收藏夹：`%LocalAppData%\GenshinBrowser\favorites.json`（最多 100 条）
- WebView2 数据：`%LocalAppData%\GenshinBrowser\WebViewProfile\`
- 日志文件：`%LocalAppData%\GenshinBrowser\logs\yyyy-MM-dd.log`

### 性能优化
- 增量列表更新：搜索时只更新变化的项，避免全量刷新
- MVVM 控制面板：控制窗口状态、命令、列表过滤集中到 ViewModel，减少界面与浏览器逻辑耦合
- 防抖保存：配置文件 500ms 防抖，减少 I/O 操作
- 异步退出：后台保存配置，立即关闭窗口
- UTC 时间存储：统一使用 UTC，避免时区问题

## 🛠️ 技术栈

- **UI 框架**：WPF (.NET 8)
- **浏览器内核**：Microsoft Edge WebView2
- **配置存储**：JSON 文件
- **全局热键**：Win32 API (SetWindowsHookEx)

## 📝 说明

- 由于使用全局 `K` 键监听，在任意软件输入 `K` 都会触发播放控制
- 管理员权限可以提高全局监听的生效概率，但不能保证对所有反作弊或独占输入场景都 100% 有效
- 播放控制通过执行页面内脚本查找 `video` 标签实现，B 站播放器结构大改时可能需要微调脚本
