# Genshin Browser

一个专门给 B 站原神攻略视频用的 Windows 浮窗浏览器，基于 `WPF + WebView2`。

## 当前功能

- 固定复用 WebView2 profile，保留 B 站登录态、Cookie 和缓存
- 默认恢复上次退出时的页面
- 默认恢复上次自由模式下的窗口位置和大小
- 主浏览器窗口只负责显示网页
- 地址栏、模式切换、播放控制、浏览记录都放在单独的控制窗口
- 控制窗口只在自由模式显示
- 控制窗口会记住自己的位置和大小
- 支持收藏当前页面，并在控制窗口里快速打开收藏夹
- 固定模式保持当前位置并置顶，不自动挪动窗口
- `F8` 在固定模式和自由模式之间切换
- 全局监听裸 `K` 键，直接控制当前页面视频播放/暂停
- 程序清单默认要求管理员权限，尽量覆盖被游戏抢占输入时的情况

## 运行要求

- Windows 10/11
- 已安装 Edge WebView2 Runtime
- 已安装 .NET 8 SDK 或兼容的 Visual Studio 构建环境

## 发布

- 自包含发布脚本: `release/publish-win-x64.ps1`
- 默认发布目录: `dist/win-x64-self-contained/`
- 自包含发布不要求目标机器单独安装 .NET，但仍需要 `WebView2 Runtime`
- 发布目录内附带 `README.txt`，可直接随压缩包分发

## 归档

- `archive/single-window-v1/`: 原来的单窗口版本源码快照

## 说明

- 由于使用全局裸 `K` 监听，在任意软件输入 `k` 都会触发播放控制，这是当前按你的偏好实现的行为。
- 管理员权限可以提高全局监听在部分场景下的生效概率，但不能保证对所有反作弊或独占输入场景都 100% 有效。
- 播放控制通过执行页面内脚本查找 `video` 标签实现，B 站播放器结构大改时可能需要微调脚本。
